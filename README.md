
# WJPhotoBrowser
* An easy way to browse image
* 用法简单的图片浏览器，可以浏览网络图片和本地图片

* 内部使用UICollectionView实现，摒弃一般第三方常用的scrollView添加子视图循环利用的方式，滑动流畅无比。个人比较信任苹果的工程师，因此觉得collection view 实现方式性能更好。
